# Callouts

Callouts use a quiet surface with a small semantic accent. They should add
structure without overpowering the page.

!!! note "Note"
    Use notes for supporting context.

!!! tip "Tip"
    Use tips for practical next steps.

!!! warning "Warning"
    Use warnings for temporary limitations or risky setup details.

!!! danger "Danger"
    Use danger only when the reader should stop and check something first.

??? info "Collapsed callout"
    Collapsed callouts keep the icon, title, and chevron aligned.
