# Tables

Tables use hairline rules, muted headers, and a subtle row hover. They avoid
zebra striping by default.

## Token Reference

| Token | Light | Dark |
| --- | --- | --- |
| `--background` | `oklch(1 0 0)` | `oklch(0.145 0 0)` |
| `--foreground` | `oklch(0.145 0 0)` | `oklch(0.985 0 0)` |
| `--muted` | `oklch(0.97 0 0)` | `oklch(0.205 0 0)` |
| `--border` | `oklch(0.92 0 0)` | `oklch(1 0 0 / .1)` |
| `--ring` | `oklch(0.708 0 0)` | `oklch(0.556 0 0)` |
