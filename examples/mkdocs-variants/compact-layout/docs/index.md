# Compact Layout Fixture

This fixture narrows the content and sidebar widths. It is useful for catching
overflow around page actions, tabs, and right-side navigation.

=== "TOML"

    ```toml
    [project.extra.alloy]
    content_width = "40rem"
    sidebar_width = "12rem"
    ```

=== "YAML"

    ```yaml
    extra:
      alloy:
        content_width: "40rem"
        sidebar_width: "12rem"
    ```
